Hierarchy Separator

This Unity library will help you create aesthetic hierarchy separators for your Unity scene inside the Editor.

You can add it to your project using the option Install package from git URL. Source code and package at https://github.com/gavwood/HierarchySeparator

Together Porting

Together Porting was a demonstration at Interactivity showcasing locomotion techniques for social VR, specifically designed to improve group interactions in virtual environments. Read about it here: https://github.com/gavwood/togetherporting.html
